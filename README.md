# Recipe Gallery

This project is a React application built with Vite. It displays a curated gallery of recipes using static data and demonstrates React list rendering with the `map()` method.

## Features

- Displays recipe cards with title, image, and ingredients
- Uses reusable React component structure
- Uses `map()` to render lists
- Includes test cases with Vitest and React Testing Library
- Responsive layout with CSS Grid

## Technologies Used

- React
- Vite
- CSS
- Vitest
- React Testing Library
